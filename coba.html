Looping Data JSON di HTML Native

Contoh cara membuat loop untuk menampilkan elemen HTML dari data JSON, termasuk cara membuat struktur JSON yang sesuai.

1. Struktur Data JSON

Berikut contoh struktur JSON yang berisi data produk:

// Format JSON (JavaScript Object Notation) { "produk": [ { "id": 1, "nama": "Laptop ASUS VivoBook", "kategori": "Elektronik", "harga": 8500000, "stok": 12, "deskripsi": "Laptop dengan prosesor Intel i5 dan RAM 8GB" }, { "id": 2, "nama": "Smartphone Samsung Galaxy S21", "kategori": "Elektronik", "harga": 10500000, "stok": 8, "deskripsi": "Smartphone dengan kamera 64MP dan baterai 4000mAh" }, { "id": 3, "nama": "Meja Kantor", "kategori": "Furniture", "harga": 1250000, "stok": 15, "deskripsi": "Meja kayu minimalis untuk kantor" }, { "id": 4, "nama": "Buku Pemrograman JavaScript", "kategori": "Buku", "harga": 150000, "stok": 25, "deskripsi": "Buku panduan lengkap pemrograman JavaScript" }, { "id": 5, "nama": "Kursi Ergonomis", "kategori": "Furniture", "harga": 1850000, "stok": 7, "deskripsi": "Kursi kantor yang nyaman untuk postur tubuh" } ] }

Catatan: JSON harus valid dengan tanda kutip ganda untuk kunci dan string. Array ditandai dengan kurung siku [], objek dengan kurung kurawal {}.

2. Menampilkan Data dengan Looping di HTML

Data JSON di atas akan ditampilkan dalam kartu produk menggunakan JavaScript:

// JavaScript untuk melakukan looping data JSON function tampilkanProduk(produkArray) { const container = document.getElementById('produkContainer'); container.innerHTML = ''; // Looping melalui setiap item dalam array produkArray.forEach(produk => { // Membuat elemen HTML untuk setiap produk const card = document.createElement('div'); card.className = 'card'; // Format harga ke Rupiah const hargaFormatted = new Intl.NumberFormat('id-ID', { style: 'currency', currency: 'IDR', minimumFractionDigits: 0 }).format(produk.harga); // Mengisi konten kartu card.innerHTML = `

${produk.nama}

Kategori: ${produk.kategori}

Harga: ${hargaFormatted}

Stok: ${produk.stok} unit

Deskripsi: ${produk.deskripsi}

ID: ${produk.id}

`; // Menambahkan kartu ke container container.appendChild(card); }); }

3. Cara Membuat JSON

JSON dapat dibuat dengan beberapa cara:

// 1. Membuat JSON secara manual sebagai string const jsonString = `{ "produk": [ { "id": 1, "nama": "Contoh Produk", "harga": 100000 } ] }`; // 2. Membuat objek JavaScript lalu mengonversinya ke JSON const dataObjek = { produk: [ { id: 1, nama: "Contoh Produk", harga: 100000 } ] }; // Konversi objek ke string JSON const jsonDariObjek = JSON.stringify(dataObjek, null, 2); // 3. Membaca JSON dari file eksternal fetch('data.json') .then(response => response.json()) .then(data => { console.log(data); // Lakukan sesuatu dengan data }) .catch(error => console.error('Error:', error));

Tips: Gunakan JSON.stringify(objek, null, 2) untuk mengonversi objek JavaScript ke JSON dengan indentasi 2 spasi agar mudah dibaca.
